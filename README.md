# Links Hub

## Cara jalankan lokal
```bash
# Clone Repository
git clone https://github.com/ShinWolf-Subject/NueveLabs.git

# Masuk ke direktory proyek
cd NueveLabs

# Install dependensi
npm Install

# Jalankan server lokal
npm run dev
```

Server jalan di:
```
 > http://localhost:5173   # Default Vite
```
